About

LuckLick 3rd Anniversary Website

Web / Event / Responsive

犬カフェ「LuckLick」の3周年記念イベントに合わせて制作した特設サイト。
店舗でのフォトコンテスト企画に合わせ、応募・店頭投票・結果発表までの流れが視覚的に伝わるデザインを意識しました。
スマホからのアクセスを前提にUIを最適化し、店内ポスターのQRコード経由で誘導される動線も設計しました。

担当
